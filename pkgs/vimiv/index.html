vimiv

An image viewer with vim-like keybindings

Vimiv is an image viewer with vim-like keybindings. It is written in python3 using the Gtk3 toolkit. Some of the features are: - Thumbnail mode - Simple library browser - Basic image editing - Command line with tab completion
