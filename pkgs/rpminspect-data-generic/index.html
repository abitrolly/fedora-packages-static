rpminspect-data-generic Subpackage of rpminspect

Template data files used to drive rpminspect tests

  • Bodhi

The rpminspect-data-generic package is meant as a template to build your product's own data file. The files in it contain product-specific information. The files in this package explain how to construct the control files.
