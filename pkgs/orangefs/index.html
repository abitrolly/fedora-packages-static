OrangeFS (formerly PVFS2) is a high-performance parallel network file system designed for use on high performance computing systems. It provides very high performance access to disk storage for parallel applications. It is accessible through a variety of interfaces, including the native OrangeFS library, the kernel, FUSE, and MPI-IO. This package provides the pvfs2-client-core which is required to use the kernel module.
