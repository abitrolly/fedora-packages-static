ktp-accounts-kcm

KDE Configuration Module for Telepathy Instant Messaging Accounts

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This is a KControl Module which handles adding/editing/removing Telepathy Accounts. It interacts with any Telepathy Spec compliant AccountManager to manipulate the accounts.
