fedora-chromium-config

Fedora customizations for Chromium/Chrome

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

This package is used to install customizations for Chromium/Chrome that are recommended by Fedora, including a user-agent string identifying the system as Fedora. It includes a GSSAPI configuration that enables access to many Fedora Project services. To add support for other domains, replace the symlink /etc/chromium/policies/managed/00_gssapi.json with your own content.
