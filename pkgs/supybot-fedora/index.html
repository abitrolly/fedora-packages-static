supybot-fedora

Plugin for Supybot to interact with Fedora services

  • Bodhi
  • Bugzilla

  • Koji
  • SCM

A Supybot plugin which provides access to Fedora information. Implements a variety of commands, such as: * fas * fasinfo * ext * bug * whoowns These provide various information from the Fedora Package Database and Account System and provide it via IRC
