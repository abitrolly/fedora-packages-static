perl-Sub-Delete

Perl module to delete subroutines

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

Sub::Delete provides one function, delete_sub, that deletes the subroutine whose name is passed to it. (To load the module without importing the function, write use Sub::Delete();.) This does more than simply undefine the subroutine in the manner of undef &foo, which leaves a stub that can trigger AUTOLOAD (and, consequently, won't work for deleting methods). The subroutine is completely obliterated from the symbol table (though there may be references to it elsewhere, including in compiled code).
