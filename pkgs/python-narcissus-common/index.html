This package contains the commons components for narcissus. Narcissus is a web application that visualizes web server hits as they happen in real time. Features: * IP addresses converted to latitude/longitude, then streamed via WebSockets to `polymaps `_. * Realtime graphs of what countries are downloading what content with `d3 `_. * `Ã˜mq (zeromq) `_ on the backend. * **Fast**. No polling. You can see a demo running live at http://narcissus.rc.rit.edu It is visualizing the logs of http://mirror.rit.edu
