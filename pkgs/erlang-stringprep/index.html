erlang-stringprep

A framework for preparing Unicode strings to help input and comparison

  • Bodhi
  • Bugzilla
  • FAF
  • Koji
  • SCM

Stringprep is a framework for preparing Unicode test strings in order to increase the likelihood that string input and string comparison work. The principle are defined in RFC-3454: Preparation of Internationalized Strings. This library is leverage Erlang native NIF mechanism to provide extremely fast and efficient processing.
