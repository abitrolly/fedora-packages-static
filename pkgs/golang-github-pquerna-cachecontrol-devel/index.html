Cachecontrol implements RFC 7234 Hypertext Transfer Protocol (HTTP/1.1): Caching. It does this by parsing the Cache-Control and other headers, providing information about requests and responses -- but Cachecontrol does not implement an actual cache backend, just the control plane to make decisions about if a particular response is cachable.
